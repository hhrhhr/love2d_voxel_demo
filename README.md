# control
|key| action |
|-|-|
| `W`/`S` | forward/backward |
| `A`/`D` | strafe left/right |
| `Up`/`Down` | change height |
| `Left`/`Right` | rotate |
| `KP+`/`KP-` | change far plane |
| `PageUP`/`PageDown` | change near plane |
| `Home`/`End` | change horizon line |
| `Ins`/`Del` | change FOV |
| `KP*`/`KP/` | change LOD |
| `0` | toggle line/dot view |
| `m` | load random map|
